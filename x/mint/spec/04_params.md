# Parameters

The minting module contains the following parameters:

| Key                    | Type   | Example                     |
|------------------------|--------|-----------------------------|
| block_reward           | coin   | {amount: 1, denom: "stake"} |
| max_halving_periods    | uint32 | 7                           |
| current_halving_period | uint32 | 2                           |
| halving_blocks         | uint64 | 20000                       |
